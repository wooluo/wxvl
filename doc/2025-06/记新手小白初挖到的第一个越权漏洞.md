# 渗透思路  
  
在遇到有预约功能的页面时，不妨尝试一下是否可以越权取消其他用户的预约。  
# 渗透过程  
  
作为一个刚刚入门的小白，某日在挖了一整天漏洞毫无收获后正准备放弃，面对着某高校校内医院小程序中有着预约功能的页面，突然灵机一动，既然能预约和取消预约，为何不试试能不能取消别人的预约呢，说干就干。
