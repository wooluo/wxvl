# 漏洞说明  
  
某供应商运维安全管理系统存在命令执行漏洞，该系统由业内多家厂商采购贴牌/二开。该漏洞是由于运维安全管理系统设备接口请求处理存在缺陷，参数未经过严格校验导致的。攻击者可利用该漏洞在设备上执行任意系统命令，写入WebShell，获取设备控制权限。
