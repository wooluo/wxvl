什么是 MCP？  
  
  
Model Context Protocol (MCP)  
 是一种用于连接 LLM 与外部环境（工具、代码、数据等）的开放协议。它通过结构化的上下文定义（Context）与标准接口调用（Function Calls / Tools），允许模型「理解」自身所处的上下文，并「调用」外部能力完成复杂推理与自动化任务。
